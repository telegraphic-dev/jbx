# Documentation

Use this site for the installed CLI: install steps, command behavior, examples, schemas, and safe automation patterns.

- [Command reference](/docs/commands/) gives every command its own page.

## Core loop

```bash
jbx skill list
jbx skill get jbx-check
jbx check src --json
jbx doctor --json
```
